How to install in Gmail

  1. Click inside the white preview box above, then Cmd+A to select all within it
  2. Copy (Cmd+C)
  3. Open Gmail → Settings (gear) → See all settings → General
  4. Scroll to "Signature" → click "+ Create new" → name it "Firmcraft"
  5. Paste (Cmd+V) into the editor
  6. Set as default for "New emails" and "Reply/forward"
  7. Scroll down → Save Changes
